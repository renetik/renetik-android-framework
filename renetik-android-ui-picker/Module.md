# Module renetik-android-ui-picker

A number/value picker wheel built on ShawnLin013/NumberPicker.

# Package renetik.android.ui.picker

`CSNumberPicker` — a picker wheel exposing selection as a `CSProperty<Int>`
(`index`) and a scroll event, with `load(data, selected)`, two-way
`index(property)` binding, and `min`/`max`/`circulate` configuration.
