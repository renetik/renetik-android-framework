# Module renetik-android-json

JSON mapping for Kotlin objects, lists and maps, with a lightweight
`CSJsonObject` model used by the store and preset modules.

# Package renetik.android.json

`toJson` / `parseJson` / `parseJsonMap` / `parseJsonList` conversions and the
`CSJson` configuration (e.g. `forceString`, pretty printing).

# Package renetik.android.json.obj

`CSJsonObject` — a mutable JSON-backed object with typed getters/setters and
round-trip serialization.
