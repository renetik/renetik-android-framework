# Module renetik-android-material

Material Components integration: CS slider views and extensions binding Material
widgets to framework properties. Requires a Material Components theme.

# Package renetik.android.material

`CSSlider`, `CSRangeSlider`, `CSVerticalSlider`; two-way `Slider` property
binding (`value(property, min, max, step)`); `TextInputLayout` form helpers
(`inputLayout`, `withStartIcon`, `withStartIconClear`); and dialog, chip,
snackbar and progress extensions.
