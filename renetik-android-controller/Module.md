# Module renetik-android-controller

View-controller layer: lifecycle-aware views, navigation and paging on top of a
single Activity.

# Package renetik.android.controller.base

`CSView<V>` and `CSActivityView<V>` — lifecycle-aware view wrappers with
parent/child destruction, registrations and activity resume/pause/back hooks.

# Package renetik.android.controller.navigation

`CSNavigationView` / `CSNavigationItemView` — push/pop navigation with
animations and dialogs inside one Activity.

# Package renetik.android.controller.view

Data-driven view helpers: `viewFactory`/`addView` list-to-views binding,
`CSGridView`, and `CSWrapHeightRecyclerView`.
