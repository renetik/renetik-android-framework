# Module renetik-android-imaging

Image loading and processing built on Glide. Supplies the Glide-backed
implementation of the ui module's `CSImaging` abstraction.

# Package renetik.android.imaging

`CSGlideImaging` plus `File`/`Uri`/`Bitmap` extensions: `write`, `loadBitmap`,
`resizeImage`, `scale`, EXIF orientation fix, and `BorderBitmapTransformation`.
